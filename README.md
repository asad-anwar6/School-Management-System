# 🎓 School Management System

A responsive multi-page School Management System interface developed as an academic project using HTML, CSS, Bootstrap, and JavaScript.

## 📌 Overview

This project provides a user-friendly interface for managing different school activities through separate dashboards for administrators, teachers, and students.

> **Note:** This is a frontend-only academic project. Backend functionality and database integration are planned for future versions.

---

## ✨ Features

- Admin Dashboard
- Student Dashboard
- Teacher Dashboard
- Login Page
- About Page
- Contact Page
- Events Management
- User-Friendly Navigation
- Responsive Design
